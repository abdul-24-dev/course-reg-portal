HOW TO DOWNLOAD FACE-API MODELS
================================

Since the automatic download in PHP isn't working reliably, follow these steps:

METHOD 1: Using PowerShell (Recommended)
---------------------------------------
1. Open File Explorer and go to: C:\xampp\htdocs\course-reg-portal
2. Right-click on "download_models.ps1"
3. Select "Run with PowerShell"
4. Wait for all files to download
5. When done, press any key to close

METHOD 2: Manual Download
-------------------------
1. Create folder: C:\xampp\htdocs\course-reg-portal\face-models
2. Download these 7 files from GitHub:

   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/tiny_face_detector_model-weights_manifest.json
   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/tiny_face_detector_model-shard1
   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/face_landmark_68_model-weights_manifest.json
   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/face_landmark_68_model-shard1
   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/face_recognition_model-weights_manifest.json
   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/face_recognition_model-shard1
   https://raw.githubusercontent.com/justadudewhohacks/face-api.js/master/weights/face_recognition_model-shard2

3. Save each file to: C:\xampp\htdocs\course-reg-portal\face-models

VERIFY DOWNLOAD
---------------
After downloading, open: http://localhost/course-reg-portal/download_models.php
Click "Verify Downloads" to check if all files are present.

NOTES
-----
- File sizes should be roughly:
  * tiny_face_detector_model-shard1 (~200 KB)
  * face_landmark_68_model-shard1 (~400 KB)
  * face_recognition_model-shard1 (~4 MB)
  * face_recognition_model-shard2 (~4 MB)
  * Other files are small JSON manifest files

- Once downloaded, the face recognition will work offline!
